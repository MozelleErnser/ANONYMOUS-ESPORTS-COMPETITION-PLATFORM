# Anonymous Esports Competition - Quick Start Guide

Welcome to the Anonymous Esports Competition Platform Challenge! This document guides you through the competition and directs you to relevant documentation.

## üìã What is This Competition?

The Anonymous Esports Competition Platform is looking for developers to enhance, extend, and optimize a privacy-preserving voting platform built with Fully Homomorphic Encryption (FHE) technology.

**Prize Pool**: $50,000

**Deadline**: December 31, 2025 (23:59 AoE)

## üöÄ Quick Start (5 Minutes)

### 1. Understand the Challenge

Read these in order:

1. **[COMPETITION-DESCRIPTION.md](./COMPETITION-DESCRIPTION.md)** (15 min)
   - Competition overview
   - Prize breakdown
   - Judging criteria

### 2. Set Up Your Development Environment

```bash
# Clone the repository
git clone https://github.com/your-username/AnonymousEsports.git
cd AnonymousEsports

# Follow the setup guide
# See: DEVELOPMENT-SETUP.md
npm install
npm run compile
npm run test
```

### 3. Choose Your Category

Pick one or more of these:

| Category | Focus | Best For |
|----------|-------|----------|
| **A** | Smart Contracts | FHE experts, Solidity developers |
| **B** | Frontend | UI/UX designers, React developers |
| **C** | Infrastructure | DevOps, automation experts |
| **D** | Education | Technical writers, educators |

See [CONTRIBUTION-EXAMPLES.md](./CONTRIBUTION-EXAMPLES.md) for examples.

### 4. Start Building

```bash
# Create feature branch
git checkout -b feature/your-feature-name

# Make your changes
# Commit regularly
git commit -m "Your meaningful commit message"

# Push to your fork
git push origin feature/your-feature-name
```

### 5. Test & Validate

Run the validation checklist before submitting:

See: [TESTING-VALIDATION-CHECKLIST.md](./TESTING-VALIDATION-CHECKLIST.md)

```bash
npm run compile
npm run test
npm run lint
npm run type-check
```

### 6. Submit

Follow the submission guidelines:

See: [SUBMISSION-GUIDELINES.md](./SUBMISSION-GUIDELINES.md)
